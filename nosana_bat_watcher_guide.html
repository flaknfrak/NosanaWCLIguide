Create Your .bat File
1.
Copy This Exact Script
Create a new .bat file and paste this script exactly as shown:

@echo off title Nosana [NODE_NUMBER] [GPU_TYPE] Loc [LOCATION] echo Nosana [NODE_NUMBER] [GPU_TYPE] Loc [LOCATION] Monitor echo Connecting to Nosana [NODE_NUMBER] [GPU_TYPE] Loc [LOCATION]... echo Server: [USERNAME]@[IP_ADDRESS] ssh -t -o ServerAliveInterval=15 -o ServerAliveCountMax=3 "[USERNAME]@[IP_ADDRESS]" "docker logs -f --tail=10 nosana-node" | findstr /i "QUEUED found claimed starting started completed running" | findstr /v "FINDSTR: Line" pause

Script Breakdown
2.
Understanding Each Line
What each line in your .bat file does:
  • @echo off - Hides command output for cleaner display
  • title - Sets window title to identify your node
  • echo [messages] - Displays connection info and status
  • ssh -t - Connects with interactive terminal
  • ServerAliveInterval=15 - Sends keepalive every 15 seconds
  • ServerAliveCountMax=3 - Disconnects after 3 failed pings
  • docker logs -f --tail=10 - Shows last 10 log lines and follows new ones
  • findstr /i - Filters for job status keywords (case insensitive)
  • pause - Keeps window open when finished

Using the Watcher
3.
Running Your Monitor
How to use your .bat watcher:
  • Double-click your .bat file to start monitoring
  • Enter SSH password when prompted (if not using keys)
  • Watch real-time job status updates scroll by
  • Look for: QUEUED ‚Üí FOUND ‚Üí CLAIMED ‚Üí STARTING ‚Üí STARTED ‚Üí RUNNING ‚Üí COMPLETED
  • Press Ctrl+C to stop monitoring
  • Press any key to close the window
The watcher filters and shows only the important job status updates, making it easy to monitor your node's activity at a glance.
