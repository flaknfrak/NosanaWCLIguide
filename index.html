üåê
Configure WSL2 DNS Settings
‚Ä¢ Configure reliable DNS resolution
‚Ä¢ Ensure stable internet connectivity in Ubuntu
‚Ä¢ Update system packages to latest versions
5.
Configure WSL2 DNS
Open WSL2 configuration file and prevent auto-generation of DNS settings:
sudo nano /etc/wsl.conf
[boot] systemd=true [network] generateResolvConf = false
Save with Ctrl + X, then Y, then Enter
6.
Set Google DNS
Configure DNS resolution:
sudo nano /etc/resolv.conf
Delete ALL existing content and replace with:
nameserver 8.8.8.8 nameserver 8.8.4.4
7.
Restart WSL2 & Test DNS
From PowerShell (as Administrator):
wsl --shutdown
Open Ubuntu and test:
nslookup google.com
Expected: Server should show 8.8.8.8
8.
Update Ubuntu System
Download latest packages and security updates:
sudo apt update && sudo apt upgrade -y
