This guide helps you create a monitoring script to watch your Nosana node logs from another Windows computer
1.
Check Windows Username
Find your Windows username on the monitor computer
whoami
Save this username - you'll need it for the batch file
2.
Get Nosana Node IP Address
Find the IP address of your Nosana node computer for Docker logs
ipconfig
Look for the IPv4 address under your active network adapter. This should be your Nosana node's IP address.
3.
Create Monitoring Batch File
Create a .bat file to automatically connect and monitor your Nosana node

Why Create a Batch File?

‚Ä¢ Automates the SSH connection process

‚Ä¢ Provides a professional countdown and connection display

‚Ä¢ Automatically runs Docker logs command to monitor your node

‚Ä¢ Easy one-click monitoring from your monitor computer

‚Ä¢ Shows real-time Nosana node activity and status

Steps to create the batch file:
1. Open Notepad
2. Copy and paste the code below
3. Replace "(windows user)" with your actual username
4. Replace "10.0.0.299" with your Nosana node's IP address
5. Save as "nosana-monitor.bat" (change file type to "All Files")
@echo off title Nosana 5 3080 Loc 5 echo Nosana 5 3080 Loc 5 Monitor echo. echo Connecting in: for /l %%i in (3,-1,1) do ( echo %%i... timeout /t 1 >nul ) echo. echo Connecting to Nosana 5 3080 Loc 5... echo Server: "(windows user)"@10.0.0.299(Vethernet) echo When prompted echo. ssh "(windows user)@10.0.0.299(Vethernet)" "docker logs -f nosana-node" pause
4.
Customize Your Script
Update the batch file with your specific details
Replace these placeholders in your batch file:
‚Ä¢ "(windows user)" ‚Üí Your actual Windows username from step 1
‚Ä¢ "10.0.0.299" ‚Üí Your Nosana node's IP address from step 2
‚Ä¢ "Nosana 5 3080 Loc 5" ‚Üí Your custom node name/description
5.
Save the Batch File
Save your customized script
Saving instructions:
1. In Notepad, click File ‚Üí Save As
2. Change "Save as type" to "All Files (*.*)"
3. Name it "nosana-monitor.bat"
4. Save to your Desktop for easy access
6.
Test Your Monitoring Script
Double-click your batch file to test the connection
What you should see:
‚Ä¢ 3-second countdown
‚Ä¢ Connection details display
‚Ä¢ SSH password prompt
‚Ä¢ Live Docker logs streaming from your Nosana node
Make sure SSH is set up on your Nosana node first (see Windows SSH guide)
